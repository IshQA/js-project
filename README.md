# js-project: Car Purchase Form

This is a simple web application that allows users to browse through a selection of cars and purchase them using a form. The application displays a list of cars with details such as brand, engine power, production year, color, and price. Users can select additional options for the car and choose a date of delivery.

## Features

- Browse through a selection of cars with detailed information.
- Select additional options for the chosen car.
- Choose a date of delivery (limited to the next 14 days).
- Submit a form to complete the purchase.

## Usage

1. Clone the repository

2. Open the `index.html` file in your web browser.

3. Browse through the list of cars, select options, choose a date, and fill in the required information in the form.

4. Click the "Submit" button to complete the purchase.

## Development

This project is built using HTML, CSS, and JavaScript. It demonstrates basic DOM manipulation techniques to create a dynamic user interface.
